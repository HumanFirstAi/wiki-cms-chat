- React 18

## Setup

Install dependencies and run:

    npm install
    npm run dev

Build for production:

    npm run build

## Usage

1. Open http://localhost:5173
2. Type a topic or question (e.g., "Apollo 11")
3. Press Enter or click Send
4. View Wikipedia article summaries with source links
